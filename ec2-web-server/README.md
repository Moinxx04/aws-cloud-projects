# EC2 Web Server Deployment

## Overview
This project demonstrates deploying a web server on an **EC2 instance**.

## What I did
- Launched an EC2 instance (Ubuntu)
- Connected via SSH
- Installed Apache web server
- Configured security group for HTTP (port 80)
- Hosted a sample webpage

## AWS Services Used
- Amazon EC2
- Security Groups
- Key Pairs

## Outcome
A public web server running on EC2, accessible via public IP.

## Key Learning
- How EC2 works as a virtual server
- Role of security groups
- Difference between SSH and HTTP ports
