# S3 Static Website Hosting

## Overview
This project demonstrates hosting a static website using **Amazon S3**.

## What I did
- Created an S3 bucket
- Disabled Block Public Access
- Enabled Static Website Hosting
- Uploaded HTML files
- Configured bucket policy for public access

## AWS Services Used
- Amazon S3
- IAM (for permissions)

## Outcome
Successfully hosted a public static website accessible via an S3 website URL.

## Key Learning
- Difference between private and public S3 buckets
- How S3 can be used without servers
