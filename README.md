# AWS Cloud Projects

This repository contains hands-on cloud projects built using AWS services as part of my cloud learning journey.

## Projects Included

### 1. S3 Static Website Hosting
- Hosted a static website using Amazon S3
- Configured bucket policy and public access
- Enabled static website hosting

### 2. EC2 Web Server
- Deployed an Apache web server on EC2
- Configured security groups for SSH and HTTP
- Hosted a custom HTML page

### 3. EC2 + RDS Integration
- Connected EC2 to an RDS MySQL database
- Created database and tables
- Performed CRUD operations

## Technologies Used
- AWS EC2
- AWS S3
- AWS RDS
- AWS CloudWatch
- Linux (Ubuntu)

## Notes
No credentials, keys, or sensitive data are stored in this repository.
